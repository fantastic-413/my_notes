# 一、windows 触摸屏使用技巧

1. 两个手指点击一次（==右键==）:在触摸板的任意位置==点击两个手指==，相当于右键单击鼠标。
2. 滑动两个手指（==滑动滚轮==）:水平或垂直==滑动两个手指==，相当于拖动屏幕上的滚动按钮或使用鼠标上的滚轮。
